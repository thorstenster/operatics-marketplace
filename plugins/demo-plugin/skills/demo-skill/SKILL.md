# Demo Skill

Dies ist ein minimaler Beispiel-Skill zur Vorlage.

## Anleitung

Wenn dieser Skill aufgerufen wird:

1. Begrüße den Nutzer freundlich.
2. Erkläre in 1-2 Sätzen, dass dies ein Demo-Skill ist und als Vorlage für echte Skills in diesem Marketplace dient.
3. Falls Argumente übergeben wurden, gib sie zurück und erkläre kurz, wie Argumente in eigenen Skills verarbeitet werden können.

## Hinweis

Kopiere diesen Ordner (`skills/demo-skill`), um einen neuen Skill zu erstellen, und passe `SKILL.md` sowie den `name`/`description` im Frontmatter an.
